Make sure Node.js is installed in your system
 
In a new API project using Node.js, to fetch data from ByteBeam's server using API call, follow the steps given below.  
Install axios for making the API call and xlsx for generating the Excel file by entering the commands below
npm install axios xlsx fs

After installing these libraries,
make a proper JavaScript code for API call and fetching data from the server.
Then once the code is ready, 

Run the command given below in the terminal to run the JavaScript file in node environment. 
node apiKM.js
